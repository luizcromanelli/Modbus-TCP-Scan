# Modbus-TCP-Poll
Ferramenta gráfica (GUI) leve para varredura, monitoramento e diagnóstico de redes Modbus TCP. Suporta leitura de Coils, Inputs e Registers (FC01 a FC04) em formatos de 16 e 32 bits (Float/Int) com ajuste de Byte Order (ABCD a CDAB). Exibe estatísticas de tráfego (Bytes TX/RX), taxa de erro e possui tolerância a falhas industrial.
